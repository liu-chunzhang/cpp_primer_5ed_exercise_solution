(a)是错误的，因为函数体返回的结果类型是string，而函数的返回类型是int，二者不一致且不能够相互转换。程序可改为
string f() {
     string s ;
     / ... /
     return s ;
}
(b)是错误的，因为函数缺少返回值类型。如果该函数确实不需要返回任何值，则程序可改为
void f2( int i ) {
     /* ... */
}
(c)是错误的，因为函数签名里不能含有重名的形参且函数体左侧的花括号丢失了。程序可改为
int calc( int v1 , int v2 ) {
     /* ... */
}
(d)是错误的。函数体必须放在一对花括号内。程序可改为
double square( double x ) {
     return x * x ;
}
