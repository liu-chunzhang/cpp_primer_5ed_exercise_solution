(a)如果定义Final对象，则创建该对象时按照以下顺序调用构造函数
Class() ;
Base() ;
D1() ;
D2() ;
MI() ;
Class() ;
Final() ;
首先调用虚基类Base的构造函数（这样被迫提前调用了Class的构造函数），然后按照次序调用非虚函数的构造函数，即MI的构造函数（如此调用了D1的构造函数和D2的构造函数），然后是Class的构造函数，最后调用了Final的构造函数。
调用析构函数的顺序恰与此相反，即
~Final() ;
~Class() ;
~MI() ;
~D2() ;
~D1() ;
~Base() ;
~Class() ;
(b)一个Final对象只有一个Base子对象，却有两个Class子对象。
(c)(a)和(c)都是错误的，因为他们都企图用派生类的指针指向基类对象（甚至于进行赋值，基类指针不能给派生类指针赋值）；(b)也是错误的，因为其存在二义性，即Class是Final的一个二义基类（见上题(b)的分析）；(d)是正确的，因为这是派生类指针给基类指针赋值。
