溢出是一种常见的算术运算错误。因为在计算机中存储某些类型的内存空间有限，所以该类型的表示能力也是有限的，当计算的结果值超出这个范围后，就会产生未定义的数值，这种错误称为溢出。以下是三条导致溢出的表达式：
    int i = 2147483647 + 1 ;
    int j = -100000 * 300000 ;
    int k = 2015 * 2015 * 2015 * 2015 ;
我所写的这三条在定义时就溢出了，这会被予以警告。还需要注意：但是如果定义时符号大小，在运算过程中超出了，编译器不会报错，也不会警告！如：
    int p = 2147483647 ;
    ++p ;
    cout << " " << p ;
运行结果是-2147483648，并不予以报错或警告！
